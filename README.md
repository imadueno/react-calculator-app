I made this calculator app using react.

# Features

- Sum
- Subtraction
- Divison
- Multiplication
- Decimal values
- Fixed length of input of 17 digits
- Zero division don't crash the app
- Special decimal operations suported ( 1.1 + 1.2 )
- 



No libraries or dependencies neede.
